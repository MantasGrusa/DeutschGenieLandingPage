
# DeutschGenie — Landing Page

Landing page for GermanGenie, a gamified German learning app that teaches sentence structure through interactive word-tile puzzles. Built as part of the B.Sc. Web Development programme at SRH University of Applied Sciences Berlin.

**Live site:** https://mantasgrusa.github.io/DeutschGenieLandingPage/

---

## About the project

GermanGenie helps learners build real German sentences — not just memorise vocabulary. Users are presented with real-life situations and must assemble correct sentences from shuffled word tiles, covering CEFR levels A1 to C1 with adaptive difficulty.

The landing page has one goal: convert organic and paid traffic into app downloads. It uses problem-first messaging, opening with the pain points learners already feel before presenting GermanGenie as the solution.
